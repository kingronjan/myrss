# MyRSS 项目文档

## 项目概述

MyRSS 是一个基于 Vue 3 + TypeScript + Naive UI 构建的 RSS 阅读器前端应用。项目采用现代化的前端技术栈，具有响应式布局和主题切换功能。

## 项目状态

✅ **完成功能：**
- 顶栏 + 侧边栏 + 内容区域的三栏布局
- 顶部导航栏包含 myrss logo（左侧）和主题切换功能（右侧）
- 侧边栏从 `http://127.0.0.1:8000/feed/sources` 获取 RSS 源数据
- 响应式设计，支持亮色/暗色主题切换
- 侧边栏占比 10% + 内容区域占比 90% 的精确布局比例
- 项目可以成功构建 (`npm run build`)

## 技术栈

### 核心框架
- **Vue 3** - Composition API
- **TypeScript** - 类型安全

### UI 组件库
- **Naive UI** - 现代化的 Vue UI 组件库
- **@vicons/ionicons5** - 图标库
- **@vicons/utils** - 图标工具

### 状态管理与路由
- **Pinia** - 状态管理
- **Vue Router** - 路由管理

### 构建工具
- **Vite** - 现代化的前端构建工具
- **Vitest** - 测试框架
- **ESLint + Oxlint** - 代码检查
- **Prettier** - 代码格式化
- **vue-tsc** - TypeScript 类型检查

## 文件功能说明

### 核心入口文件

#### `src/main.ts`
- **功能**: 应用主入口文件
- **职责**:
  1. 初始化 Vue 应用
  2. 配置 Pinia 状态管理
  3. 集成 Naive UI 主题和全局 API
  4. 挂载应用到 DOM
- **关键特性**:
  - 创建 Naive UI 的离散 API 实例（消息、对话框、通知、加载条）
  - 将 Naive UI API 挂载到全局属性

#### `src/App.vue`
- **功能**: 应用根组件
- **职责**:
  1. 导入并渲染 MainLayout 组件
  2. 提供全局 CSS 样式
- **样式特性**:
  - 全局盒模型设置
  - 字体和抗锯齿优化

### 布局组件

#### `src/components/MainLayout.vue`
- **功能**: 主布局组件
- **职责**:
  1. 组织顶栏、侧边栏和内容区域的结构
  2. 实现 10% 侧边栏 + 90% 内容区域的精确布局
  3. 集成路由视图
- **布局特点**:
  - 使用 CSS Grid 或 Flexbox 实现精确比例布局
  - 响应式设计适配不同屏幕尺寸

#### `src/components/HeaderBar.vue`
- **功能**: 顶部导航栏组件
- **职责**:
  1. 显示 "myrss" logo（左侧）
  2. 提供主题切换功能（右侧）
  3. 集成 GitHub 链接
- **特性**:
  - 使用 Naive UI 的 NButton 组件
  - 支持亮色/暗色主题切换
  - 跟随系统主题设置

#### `src/components/Sidebar.vue`
- **功能**: 侧边栏组件
- **职责**:
  1. 从 `http://127.0.0.1:8000/feed/sources` 获取 RSS 源数据
  2. 显示 RSS 源列表
  3. 提供导航功能
- **数据获取**:
  - 使用 Fetch API 获取数据
  - 处理加载状态和错误状态
  - 响应式数据显示

### 页面视图

#### `src/views/HomeView.vue`
- **功能**: 首页视图
- **职责**:
  1. 显示应用欢迎信息
  2. 展示统计信息或功能简介
  3. 作为默认路由页面
- **特性**:
  - 使用 Naive UI 组件构建界面
  - 响应式布局

#### `src/views/AboutView.vue`
- **功能**: 关于页面视图
- **职责**:
  1. 显示项目相关信息
  2. 提供项目说明文档
- **路由配置**: 懒加载路由，优化首屏加载性能

### 路由配置

#### `src/router/index.ts`
- **功能**: Vue Router 配置
- **职责**:
  1. 定义路由路径: `/` (首页) 和 `/about` (关于页面)
  2. 配置 Web History 模式
  3. 实现路由懒加载优化
- **路由特点**:
  - 首页使用常规导入
  - 关于页面使用懒加载，生成单独的代码块

### 资源文件

#### `src/assets/base.css`
- **功能**: 基础 CSS 变量和重置样式
- **内容**:
  1. 颜色调色板（亮色/暗色主题变量）
  2. 语义化颜色变量
  3. 基础重置样式（盒模型、字体等）
  4. 暗色主题媒体查询

#### `src/assets/main.css`
- **功能**: 主样式文件
- **内容**:
  1. 导入基础样式
  2. 应用容器样式（#app）
  3. 链接和按钮样式
  4. 响应式布局规则

#### `src/assets/logo.svg`
- **功能**: 项目 logo 图片
- **使用**: 在 HeaderBar 组件中显示
